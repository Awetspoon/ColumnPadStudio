# Repository Structure

- `src/ColumnPadStudio.App` - WPF UI shell and windows
- `src/ColumnPadStudio.Application` - app contracts and exporters
- `src/ColumnPadStudio.Domain` - pure models and marker logic
- `src/ColumnPadStudio.Infrastructure` - JSON session/recovery persistence
- `tests/ColumnPadStudio.Domain.Tests` - small no-package console checks
